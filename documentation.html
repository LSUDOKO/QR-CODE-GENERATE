qrcodegeneration.py

This is the basic QR code generator script using the qrcode library with PIL (Python Imaging Library) support. It creates a simple QR code with black modules on a white background. 


import qrcode
from PIL import Image

# Create qr code instance
qr = qrcode.QRCode(
    version=1,
    error_correction=qrcode.constants.ERROR_CORRECT_L,
    box_size=10,
    border=4,
)

# Add data
qr.add_data('https://www.example.com')
qr.make(fit=True)

# Create an image from the QR Code instance
img = qr.make_image(fill_color="black", back_color="white")
img.save("qrcode.png")

Parameters Explained:

  • version: Controls the size of the QR Code (1-40). Higher values make a bigger code.
  • error_correction: Controls the error correction used for the QR Code. L (7%), M (15%), Q (25%), H (30%).
  • box_size: Controls how many pixels each "box" of the QR code is.
  • border: Controls how many boxes thick the border should be.

qrcode2.py

This script uses the segno library to create a light blue QR code with a custom border. Segno is more lightweight than the qrcode library and offers different customization options. 


import segno

# Create QR code instance
qrcode = segno.make('https://www.example.com')

# Save QR code with light blue color
qrcode.save(
    'lightblue_qrcode.png',
    scale=10,
    light='lightblue'
)

Parameters Explained:

  • scale: Controls the size of the QR code (similar to box_size in qrcode).
  • light: Sets the background color of the QR code.
  • dark: Sets the foreground color (if not specified, defaults to black).

qrcode3.py

This script creates a dark blue QR code with a custom quiet zone (border) using the segno library. 


import segno

# Create QR code 
qrcode = segno.make('https://www.example.com')

# Save with dark blue color
qrcode.save(
    'darkblue_qrcode.png',
    scale=10,
    light='white',
    dark='darkblue',
    quiet_zone=4
)

Parameters Explained:

  • quiet_zone: Controls the size of the border around the QR code.
  • dark: Sets the color of the QR code modules (data elements).

qrcode7.py

This script demonstrates how to create an animated QR code with a background image using the segno library. 


import segno
from pathlib import Path

# Create QR code
qrcode = segno.make('https://www.example.com')
out = Path('animated_qrcode.gif')

# Create animated QR code
qrcode.to_artistic(
    background='background.png',
    target=out,
    scale=10
)

Parameters Explained:

  • to_artistic: Method to create QR codes with background images or artistic effects.
  • background: The path to the background image to use.
  • target: The output file path (can be PNG, GIF, or other formats).
